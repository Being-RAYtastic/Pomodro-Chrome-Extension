# Pomodoro App

A simple web-based Pomodoro timer to boost productivity using the Pomodoro Technique.

## Features
- Visual timer display
- Notifications for session transitions
- Progressive Web App (PWA) support for offline use

## Installation
1. Clone or download the repository.
2. Open `index.html` in your web browser.

### Use it as Chrome extension
1. Go to Manage Extensions in CHROME SETTINGS and turn on DEVELOPER MODE.
2. Click on LOAD UNPACKED and select the folder
